Ein Datenbanksystem lässt sich in **drei Ebenen** Unterteilen:

![[Pasted image 20250315150217.png]]

##### Externe Schicht

Anwender nutzt Datenbanksystem mittels:
- **DQL = Data Query Language** (Datenabfragesprache)
- **DML = Data Manipulation Language** (Datenmanipulationssprache)

##### Konzeptionelle Schicht

Datenmodellbeschreibung; Strukturen, Beziehungen der Daten:
- Unterscheidung hierarchischen, vernetzten und relationalen Datenmodellen
- Einrichtung mittels **DDL = Data Definition Language**
- Sicherungsmaßnahmen mittels **TPL = Transaction Processing Language**

##### Interne Schicht

Physische Datenspeicherung
