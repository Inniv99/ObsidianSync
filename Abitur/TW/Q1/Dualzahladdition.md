Möchte man zwei Dualzahlen miteinander addieren, so muss man folgende Dinge beachten. 
$0_2 + 0_2 = 0_2$   
$1_2 + 0_2 = 1_2$ vice versa
$1_2 + 1_2 = 0_2$ aber eine $1_2$ wird auf die nächste Zeile übertragen
